3 Screen App

ENGR 116 Midterm

My first coding project that I made in my first year coding course to skip the midterm. Most of the project was created using open source code and combining it using react native and javascript. It features a 3 tab navigation with a calculator, a fixed location on google maps, and a simple game. It was able to be used with an emulator on andriod studio, in a web browser, or on a phone using the Expo app.
